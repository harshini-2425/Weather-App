🌦️ Weather App

This is a simple and responsive Weather Application built while learning React.js.
The app allows users to search for any city and view real-time weather information using the OpenWeather API.

🚀 Features
Search weather by city name
Displays:
Temperature
Weather condition
Humidity
Wind speed
Responsive UI
Clean and modern design using Tailwind CSS
🛠️ Technologies Used
React.js
Tailwind CSS
OpenWeather API
JavaScript
HTML5 & CSS3
📚 What I Learned

Through this project, I learned:

React components and props
React state management with useState
API fetching using fetch()
Handling asynchronous data
Conditional rendering
Styling with Tailwind CSS
Working with external APIs
🔗 API Used
OpenWeather API
▶️ Run Locally

npm run dev
